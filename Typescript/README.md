# ts带来的好处

- 开发大型应用更友好；在编写库，框架以及一些企业级应用时，编写时容易出现错误，比如传一个变量，是什么类型的
- 在编写代码时会对代码进行类型检查从而避免很多线上错误；
- ts提供的类型系统可以帮助我们在写代码时提供丰富的语法提示

# 定义类型的方式

- 枚举：默认枚举具有反举功能，最终编译的结果是一个对象。
- type字面量
- interface
- import
- declare

type可以使用联合类型，interface不能使用联合类型，接口的强大在于他的继承和实现，同名接口会合并

# tips

- unknow只能赋值给any|unknown
- typeof any 的类型是string|number|symbol
